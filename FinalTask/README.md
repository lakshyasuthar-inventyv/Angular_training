# FinalTask: Capstone Project

This is a comprehensive Angular project integrating advanced frontend features and backend services.

## Features

- Backend integration (see `backend/` folder for API and server code)
- Advanced routing and route guards
- Notification system using RxJS and Angular Material
- Account settings with live UI updates using signals
- Production-ready build and testing setup

## Setup

```bash
cd FinalTask
npm install
ng serve
```

### Backend Setup

To start the backend server:

```bash
cd backend
npm install
npm start
```

## Usage

- Explore advanced routing and guards for secure navigation
- Test notification system and account settings page
- Review backend integration for full-stack development
