# Task-1: Angular Basics

This project covers foundational Angular concepts and features for beginners and intermediate learners.

## Features

- Data binding: property, event, and two-way binding
- Component communication: `@Input`, `@Output`, and `EventEmitter`
- RxJS basics: `Subject`, `BehaviorSubject` for state and notifications
- Custom and built-in pipes
- Lifecycle hooks: `ngOnInit`, `ngDoCheck`, `ngAfterViewInit`, etc.
- Shared logic and UI in the `comman` folder
- Routing and guards for navigation and security

## Setup

```bash
cd Task-1
npm install
ng serve
```

## Usage

- Generate new components, services, pipes, and guards using Angular CLI
- Practice data binding and component communication in the provided examples
- Explore RxJS usage for reactive state and notifications
- Review the `comman` folder for reusable code and UI
