# Task-2: Advanced Rendering & Signals

This project demonstrates advanced Angular rendering techniques, including signals, zoneless change detection, and standalone components.

## Features

- Angular signals for reactive state management
- Zoneless rendering and manual change detection
- Standalone components and CommonModule usage
- Advanced RxJS patterns for async data
- Angular Material UI integration

## Setup

```bash
cd Task-2
npm install
ng serve
```

## Usage

- Switch between tabs to see different rendering strategies
- Explore signal-based state and manual change detection
- Review standalone component structure and CommonModule imports
